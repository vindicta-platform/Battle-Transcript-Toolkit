# Battle Transcript Toolkit

**Parse, analyze, and learn from match transcripts.**

Tools for extracting insights from WARScribe transcripts.

## Features

- **Transcript Parser** — Parse WARScribe format
- **Analytics** — Action frequency, win patterns
- **Export** — JSON, CSV, Markdown

## Installation

```bash
uv pip install git+https://github.com/vindicta-platform/Battle-Transcript-Toolkit.git
```
