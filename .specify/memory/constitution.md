## Core Principles

### I. Schema Fidelity
All transcripts MUST conform to the WARScribe specification.

### II. Non-Destructive Parsing
Original transcript data MUST be preserved. Analysis is additive.

### III. Reproducibility
Same input MUST produce identical parsed output.
